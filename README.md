# Uilerial Studio

**Uilerial Studio** — patterns, Konva canvas, and live galleries for the [Techlestial](https://techlestial.vercel.app) studio.

| Product | URL |
|---------|-----|
| **Studio** (this repo) | [uilerial-studio.vercel.app](https://uilerial-studio.vercel.app) |
| **UI** | [uilerial.vercel.app](https://uilerial.vercel.app) |

## What it is

| Section | Path | Content |
|---------|------|---------|
| **Patterns** | `/patterns` | Hooks, context, composition, performance guides |
| **Components** | `/components` | Live Uilerial demos |
| **Canvas** | `/patterns/libraries` | Konva editors, behavior trees, drawing tools |

Legacy URLs `/concepts-lab` and `/components-lab` still work.

## Develop

```sh
npm install
npm start    # http://localhost:3333
npm run build
```
